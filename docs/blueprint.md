# **App Name**: eShop Revitalize

## Core Features:

- User Authentication: Implement secure user registration, login, and logout functionality using NextAuth and JWT/session-based authentication.
- Product Management (CRUD): Enable administrators to create, read, update, and delete products within the e-commerce platform. Includes the implementation of product schema models.
- Product Filtering and Search: Allow users to filter products based on various criteria (e.g., price, category, rating) and search for specific products by keywords.
- Image Upload to Cloudinary: Integrate Cloudinary for efficient image uploads and storage, ensuring optimized image delivery and management.
- Shopping Cart System: Develop a persistent shopping cart system that allows users to add, remove, and update items in their cart.
- Checkout Logic and Stripe Integration: Implement a secure checkout process, including integration with Stripe for payment processing (if used in original site), handling transactions, and managing orders.
- Admin Dashboard: Design and develop an intuitive admin dashboard for managing products, orders, users, and other key aspects of the e-commerce platform.

## Style Guidelines:

- Primary color: Deep Indigo (#4F3A65) to convey trust, sophistication, and modernity in the e-commerce platform's branding.
- Background color: Off-White (#F4F2F7) to provide a clean, neutral backdrop that enhances readability and visual appeal, aligning with modern UI design principles.
- Accent color: Vibrant Coral (#FF7F50) used sparingly for key interactive elements, CTAs, and highlights, drawing attention without overpowering the interface. 
- Body text: 'Inter' (sans-serif) for clean and readable UI.
- Headline text: 'Space Grotesk' (sans-serif) to add character and tech appeal. Pair with 'Inter' (sans-serif) for body text.
- Use a consistent set of minimalistic icons from libraries like Phosphor Icons to maintain a modern and uncluttered look.
- Employ a mobile-first, responsive design using Tailwind CSS grid and flexbox to ensure optimal viewing experience across various devices.
